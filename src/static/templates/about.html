About the Astro Coffee Server

{% end %} {% block pagecontent %}

astroph-coffee is a small platform to view and vote on arXiv astro-ph papers, and helps to organize paper discussions. It's a bit like VoxCharta, but much simpler, and can be fully installed on a local department server.

Code and author

The server's code is a fork of the template for coffee server established by Princeton astro alumn Waqas Bhatti and is currently hosted on Github. It is meant to be simple to install and use. Please let Remy Joseph know if something doesn't look right, or is completely broken, or if you have questions. Even better, if you encounter a problem on this website or have suggestions to improve it, feel free to open an issue via the github repo: https://github.com/herjy/astroph-coffee/issues .

Acknowledgments

astroph-coffee uses the following open-source libraries:

  • Tornado
  • SQLite3
  • Requests
  • BeautifulSoup
  • Selenium
  • pytz
  • itsdangerous
  • geoip2
  • jQuery
  • Foundation
  • MathJax

This product includes GeoLite2 data created by MaxMind, available from http://www.maxmind.com.

The astro-ph article listings are generated from the arXiv e-print server.
